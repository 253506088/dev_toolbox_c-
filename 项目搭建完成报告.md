## ⚠️ 已知问题和警告

### 可忽略的警告

1. **SQLitePCLRaw 漏洞警告**
   - 警告代码: NU1903
   - 影响: 低（本地应用）
   - 解决: 等待官方更新或使用其他 SQLite 包

2. **OpenCvSharp4 兼容性警告**
   - 警告代码: NU1701
   - 原因: 包针对 .NET Framework
   - 影响: 无（运行正常）

3. **可空引用类型警告**
   - 警告代码: CS8625, CS8601
   - 位置: IStorageService.cs
   - 影响: 无（代码规范问题）
   - 解决: 添加可空标注或关闭检查

### 需要手动配置的部分

1. **FFmpeg** - 需要手动下载并放置到 `assets/ffmpeg/`
2. **图标资源** - 需要添加工具图标
3. **主题样式** - 需要完善 WPF 主题

## 💡 使用技巧

### 开发时

```powershell
# 推荐：使用热重载模式
.\dev.ps1 watch

# 修改 XAML 或 C# 文件后自动重载，无需重启
```

### 添加新工具

1. 在对应模块创建工具类实现 `ITool`
2. 在 `App.xaml.cs` 注册 `services.AddTransient<ITool, YourTool>()`
3. 创建对应的 View 和 ViewModel
4. 运行测试

### 发布前检查

```powershell
.\dev.ps1 test                # 确保测试通过
.\performance-test.ps1        # 性能基准测试
.\publish.ps1 -Version "x.y.z"  # 打包发布
```
