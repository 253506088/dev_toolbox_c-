## 🚀 常用命令

### 开发

```powershell
.\dev.ps1              # 运行应用
.\dev.ps1 watch        # 热重载模式
.\dev.ps1 test         # 运行测试
.\dev.ps1 build        # 构建项目
.\dev.ps1 clean        # 清理项目
```

### 发布

```powershell
.\publish.ps1 -Version "1.0.0"                  # 发布 x64 版本
.\publish.ps1 -Version "1.0.0" -IncludeArm64    # 包含 ARM64
.\publish.ps1 -Version "1.0.0" -SkipTests       # 跳过测试
```

### 测试

```powershell
.\performance-test.ps1                    # 运行性能测试
.\performance-test.ps1 -CreateTestFiles   # 创建测试数据
.\performance-test.ps1 -CleanupTestFiles  # 清理测试数据
```

### 手动命令

```powershell
dotnet build                              # 构建
dotnet run --project src/DevToolbox.App   # 运行
dotnet test                               # 测试
dotnet clean                              # 清理
```

## 🎯 添加新工具的步骤

### 1. 创建工具类

在对应模块项目中创建工具类：

```csharp
// src/DevToolbox.Tools.Text/MyNewTool.cs
using DevToolbox.Core.Interfaces;

namespace DevToolbox.Tools.Text;

public class MyNewTool : ITool
{
    public string Id => "my-new-tool";
    public string Name => "我的新工具";
    public string Description => "工具描述";
    public string Icon => "ToolIcon";
    public ToolCategory Category => ToolCategory.Text;
    public int Order => 10;
}
```

### 2. 注册工具

在 `App.xaml.cs` 中注册：

```csharp
private void ConfigureServices(IServiceCollection services)
{
    // ... 其他服务
    
    // 注册工具
    services.AddTransient<ITool, MyNewTool>();
}
```

### 3. 创建工具界面（可选）

创建对应的 UserControl：

```xaml
<!-- src/DevToolbox.Tools.Text/Views/MyNewToolView.xaml -->
<UserControl x:Class="DevToolbox.Tools.Text.Views.MyNewToolView"
             xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
             xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Grid>
        <!-- 工具界面 -->
    </Grid>
</UserControl>
```


## 🎨 代码风格

- 使用 4 空格缩进
- 类名使用 PascalCase
- 方法名使用 PascalCase
- 私有字段使用 _camelCase
- 接口以 I 开头
- 异步方法以 Async 结尾

## 🧪 测试规范

- 测试类名：`{ClassName}Tests`
- 测试方法名：`{MethodName}_Should{ExpectedBehavior}_When{Condition}`
- 使用 Arrange-Act-Assert 模式
