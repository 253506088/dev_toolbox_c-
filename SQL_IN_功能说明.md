# SQL IN 功能说明

## 功能概述

SQL IN 工具可以将列表数据快速转换为 SQL IN 语句格式，方便在数据库查询中使用。

## 界面布局

应用采用左右分栏布局，效仿 JSON 格式化工具的设计风格：

- **左侧**：输入区域 - 用于输入原始列表数据
- **右侧**：输出区域 - 显示转换后的 SQL IN 语句

## 使用方法

### 1. 输入数据

在左侧输入区域中，每行输入一个值，例如：

```
apple
banana
orange
grape
```

或者数字列表：

```
1001
1002
1003
1004
```

### 2. 选择引号类型

在右侧工具栏中，可以选择三种引号类型：

- **单引号 '**：适用于字符串值（默认）
  - 输出示例：`('apple', 'banana', 'orange', 'grape')`

- **双引号 "**：适用于某些数据库的字符串值
  - 输出示例：`("apple", "banana", "orange", "grape")`

- **无引号**：适用于数字或不需要引号的标识符
  - 输出示例：`(1001, 1002, 1003, 1004)`

### 3. 转换

点击"转换 (F4)"按钮或按 F4 键，即可在右侧看到转换后的 SQL IN 语句。

### 4. 使用结果

- 点击"复制"按钮或按 Ctrl+Shift+C 将结果复制到剪贴板
- 将结果粘贴到 SQL 查询中使用：

```sql
SELECT * FROM products WHERE name IN ('apple', 'banana', 'orange', 'grape');
```

```sql
SELECT * FROM orders WHERE order_id IN (1001, 1002, 1003, 1004);
```

## 快捷键

- **F4**：转换为 SQL IN 语句
- **Ctrl+Shift+C**：复制输出结果
- **Ctrl+滚轮**：调整编辑器字体大小

## 特性

1. **自动去重空行**：会自动忽略输入中的空行
2. **引号转义**：如果值中包含引号，会自动进行转义处理
3. **实时统计**：显示输入行数和输出字符数
4. **字体大小记忆**：会记住上次设置的字体大小
5. **自动转换**：切换引号类型时，如果已有输出内容，会自动重新转换

## 使用场景

- 从 Excel 或文本文件中复制多个 ID 或名称，快速生成 SQL 查询条件
- 批量处理数据库查询时，避免手动添加引号和逗号
- 测试 SQL 查询时，快速构造 IN 子句

## 注意事项

- 输入的每一行都会被视为一个独立的值
- 空行会被自动忽略
- 如果使用引号类型，值中包含的引号会自动转义（双写）
